## Introduction

This small demo test pipeline tries to replicate a simple MCMICRO run using core tools, only using nf-core modules.

## Pipeline summary

The current test runs the following modules in succession.

1. Background subtraction
2. Segmentation
3. Quantification
4. Scimap
